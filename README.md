# APP Objective

Projeto Code Hero - Projeto responsivo para mostrar personagens de anime

## 📦 Implantação

Siga as instruções a seguir para executar o projeto.

#### 📱 Front-End
Abra o "app-objective" com o Visual Studio Code, e no terminal execute o seguinte:

```
npm install --force
```

E logo em seguida:

```
ng serve
```
